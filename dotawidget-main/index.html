🎮 MMR Tracker Widget

Профессиональный виджет для отображения MMR в OBS Studio. Поддерживает настройку игрока, отображение ранга и истории матчей.

Предварительный просмотр

Виджет автоматически обновляется и показывает текущий MMR, ранг игрока и последние результаты матчей с красивыми анимациями.

Как использовать в OBS

Шаг 1: Скопируйте URL

Введите имя игрока и скопируйте готовую ссылку:

Шаг 2: Добавьте в OBS

1. Откройте OBS Studio
2. Добавьте источник "Браузер"
3. Вставьте скопированную ссылку в поле URL
4. Установите размер: 400x150 пикселей

Шаг 3: Настройте данные

Разместите файл mmr_data.json рядом с вашим основным приложением. Виджет будет автоматически читать данные из этого файла.

Шаг 4: Контроль доступа (опционально)

Виджет поддерживает систему разрешений. Отредактируйте файл allowed_users.json для контроля доступа. Пользователи, не включенные в список, увидят MMR = 1 и сообщение "STREAMER IS NOT ALLOWED".

Возможности виджета

🏆 Автоматическое определение ранга

Виджет автоматически определяет и показывает правильную иконку ранга на основе MMR

📊 История матчей

Отображение последних 10 результатов матчей с цветовой кодировкой

📺 Интеграция с Twitch

Показывает количество зрителей на Twitch в реальном времени

🎨 Красивые анимации

Плавные переходы, эффекты свечения и анимации обновления данных
